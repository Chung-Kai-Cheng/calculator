## How to use
1.Install Node.js
```bash
Node.js v20.9.0
```
2.Clone the repository
```bash
https://github.com/njdhdl1223/Alpha-shop.git
```
3.Install npm packages
```bash
npm install
```
4.Install Sass
```bash
npm install -D sass
```
5.Launch the application
```bash
npm run dev
```
5.visit http://localhost:3000

6.``` ctrl + c ```in Terminal to stop running the project.
